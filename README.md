# BFL Site 

Лендинг для направления БФЛ компании "Есть решение".

## Что внутри

- `index.html` — основная страница лендинга
- `styles.css` — общая дизайн-система и адаптивные стили
- `script.js` — калькулятор, sticky CTA и аккордеоны
- `organization.html` — юридическая информация о компании
- `privacy.html` — политика конфиденциальности
- `consent.html` — согласие на обработку персональных данных
- `assets/img` — изображения сайта с web-friendly именами

## Как открыть

Откройте файл `index.html` в браузере или подключите репозиторий к любому статическому хостингу.

## Важные замечания

1. Контактные данные, реквизиты и актуальный логотип нужно подставить перед публикацией в рекламу.
2. Формы сейчас подготовлены на фронтенде и готовы к подключению CRM, почты или мессенджера.
3. Проект собран как статический сайт, потому что в текущей среде отсутствуют `node` и `npm`.
